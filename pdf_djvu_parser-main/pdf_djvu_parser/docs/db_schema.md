# Описание структуры базы данных (SQLite)

В проекте используется реляционная СУБД SQLite. Файл БД хранится в папке `data/parser_database.db`.

## Таблица `documents`
Предназначена для хранения метаданных обработанных файлов и извлеченного текста.

| Поле              | Тип данных    | Описание                                             |
|-------------------|---------------|------------------------------------------------------|
| `id`              | INTEGER       | Первичный ключ (PRIMARY KEY), автоинкремент.         |
| `file_name`       | TEXT          | Имя исходного файла (например, `document.pdf`).      |
| `file_format`     | TEXT          | Формат файла (`pdf` или `djvu`).                     |
| `file_size_bytes` | INTEGER       | Размер файла в байтах.                               |
| `parsed_date`     | DATETIME      | Дата и время выполнения парсинга (ISO 8601).         |
| `word_count`      | INTEGER       | Количество слов в извлеченном тексте.                |
| `text_content`    | TEXT          | Полный извлеченный текст из документа.               |
